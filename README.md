# 🎬 CineMatch - Movie Recommendation System

A content-based movie recommendation system built using:

- Python
- Streamlit
- TMDB API
- Scikit-learn
- Cosine Similarity

## 🚀 Features

- Recommend similar movies
- Recommend by genre
- Adjustable recommendation count
- Real-time poster fetching using TMDB API
- Secure API key using .env

## 🛠 Tech Stack

- Pandas
- Scikit-learn
- Streamlit
- TMDB API

## 📦 Setup

1. Clone the repository
2. Install requirements:

pip install -r requirements.txt

3. Add your TMDB API key in api.env:

TMDB_API_KEY=your_api_key_here

4. Run:

streamlit run app.pys
